# MediSlot Backend

Simple REST API for doctor appointment booking and scheduling system.

## Features
- User registration and login
- JWT authentication
- Role-based access control
- Doctor time slot creation
- Appointment booking and cancellation
- PostgreSQL integration
- Docker and Docker Compose support

## Tech Stack
- Go
- Gin
- PostgreSQL
- JWT
- Docker
- Docker Compose

## How to Run
1. Start Docker:
   docker compose up -d

2. Run the application:
   go run cmd/main.go

3. Test API using Postman
