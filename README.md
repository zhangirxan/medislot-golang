# Medislot Backend Golang Project
Medislot is a REST API for booking appointments with doctors.

## Features
- User registration and login
- JWT authentication
- Role-based access control
- Doctor time slot management
- Patient appointment booking
- PostgreSQL database
- Docker support

## Tech Stack
- Go
- Gin
- PostgreSQL
- JWT
- Docker
